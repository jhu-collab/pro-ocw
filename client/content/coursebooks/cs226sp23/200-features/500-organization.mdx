# Organization

In this step, we'll set up organizations.

## Components

Scalerepo has a number of organization components assembled for after you sign up. You can use these components to build your own organization flow.

## Accounts and Teams

Scalerepo has each account associated with a team. You can use this to manage your organization.

## Invitations

Scalerepo has a number of invitation components assembled for after you sign up. You can use these components to build your own invitation flow.
